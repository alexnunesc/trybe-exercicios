# Você será capaz de:
  1. Utilizar INNER JOIN para combinar dados de duas ou mais tabelas;

  2. Utilizar LEFT JOIN e RIGHT JOIN para combinar dados de duas ou mais tabelas;
  
  3. Utilizar SELF JOIN para fazer join de uma tabela com ela própria.

  **A ideia do JOIN é permitir combinar registros de duas ou mais tabelas, através do relacionamento que uma tabela tem com a outra.**
